What is Venux?

Venux is an innovative software company that is commited to improve software experience and fix the architectural problems in modern systems with 21st century knowledge.

What is UID

Simply put, UID is a file. This file acts as a personal wallet no one can see, steal, or use without your credentials. This file contains all of the required information to keep the rest of your data safe.

What is H.I.P.S.TM?

Venux's security system is based on the H.I.P.STM principle. It allows the user to have and store data in any location while restricting access for everyone except the authorized user.

How does Venux privacy work?

Venux heavily relies on mathematics in order to achive its goals. We don't use databases nor have an index of our users. Profile (UID) is stored somewhere within the network and updated upon login and modification. All information leaving your computer/device leaves encrypted. In order to decrypt data, login is required and can not be accessed without the correct credentials.

What remote services can I connect to with Venux?

Venux can practically work with most modern protocols and online services available on the market. They are FTP, Amazon, Azure, Box.net, WebDAV, DropBox, EDI, Google Drive, iCloud, SFTP, SkyDrive, SSH, and Yandex. There are other protocols in the works so please update regularly.

How does encryption work?

Encryption is a transformation of information by a specific algorithm using a specific key to obfuscate the original data. Please refer to http://en.wikipedia.org/wiki/Encryption for more information.

What will happen if I forget my password or user name?

You will not have access to your account. Please make sure that you remember your username and password. Venux does not store keys or passwords and will not be able to retrieve them for you.

What personal data does Venux Collect?

Our software collects information such as Venux software version, screen resolution and operating system version. This is done for statistical purposes to improve our software and make your life easier. Personal information, settings of any kind or any other information that is not related to pure software function is not collected by Venux. All of your personal data is stored in your UID which is inaccessible without login credentials.
